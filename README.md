# 📚 Taller 2 – Programación en Python

Este repositorio contiene el **Taller 2 de programación en Python**, donde se desarrollan diferentes ejercicios enfocados en fortalecer la lógica de programación, el uso de estructuras de control, funciones y manejo de estructuras de datos.

El taller está organizado por **secciones**, donde cada carpeta contiene ejercicios relacionados con un tema específico del lenguaje Python.

## 🎯 Objetivos del Taller

- Practicar la **lógica de programación**
- Comprender el uso de **condicionales**
- Aplicar **ciclos o estructuras repetitivas**
- Implementar **funciones**
- Utilizar **estructuras de datos como listas y diccionarios**
- Desarrollar un **programa funcional en consola**

---

## ⚙️ Requisitos

Para ejecutar este proyecto se necesita:
- **Python 3 o superior**
- Un editor de código (VS Code, PyCharm o IDLE)

---

## ▶️ Ejecución de los programas

Para ejecutar cualquiera de los ejercicios se debe abrir una terminal dentro de la carpeta del proyecto y ejecutar el archivo deseado.

Ejemplo:
`python seccion1/1.1.py`

También se puede ejecutar el sistema principal del taller ubicado en la sección 6.

Ejemplo:
`python seccion6/biblioteca.py`

---

## 📖 Sistema de Gestión de Biblioteca (Sección 6)

Programa en consola que simula un sistema de gestión de biblioteca. Permite administrar un catálogo de libros mediante diferentes opciones del menú.

### Funcionalidades del sistema:
- Agregar, buscar y eliminar libros.
- Mostrar lista completa y estadísticas.
- Prestar y devolver libros.
- Exportar catálogo a texto.
